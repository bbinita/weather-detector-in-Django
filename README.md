# - Weather Detector App  (Build using Django)

This is a simple weather detector that shows real-time weather details for any city, state, or country entered by the user.
---

##  Features  
- Search weather by city, state, or country  
-  Shows temperature, humidity, and country code 
-  Displays real-time weather conditions  
-  Fetches live data from OpenWeatherMap API  

---

## How It Works
1. The user enters a city, state, or country name.
2. The app sends a request to the OpenWeatherMap API.
3. The API returns real-time weather data in JSON format.
4. The app extracts and displays details like temperature, humidity, coordinates, etc.
